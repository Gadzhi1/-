# Задача проекта - автоматизация процессов, связанных с долгами и заказами от клиентов и пополнением товара в магазине.
Реализованы функции добавления, поиска, сортировки, удаления, очистки, списания, изменения, графического отображения данных и это еще не все.
Можно сохранить данные о друзьях (постоянные клиенты), чтобы не записывать их каждый раз во время добавления заказа либо долга. 
К консольному приложению прилагается телеграм бот, который дублирует все основные действия в чат в удобном для чтения и поиска формате,
а также может выгрузить данные по запросу, плюс еще некоторые функции, которые не так важны, но они есть)
Для того, чтобы все работало, нужно вставить свой путь для файлов в началe файла functions.py и после этого можно собрать в приложение .exe файлы main.py и telegram.py.
